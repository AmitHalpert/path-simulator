# path-simulator



**path-simulator** is webapp for visualizing the pathfinding algorithms in real-time. <br>

### [LIVE DEMO](https://amithalpert.github.io/path-simulator/)



<img src="./path.gif">



<br>

## Algorithms

* Dijkstra
* DFS
* BFS
* A*

<br>

## Build Guide
```bash
git clone https://github.com/AmitHalpert/path-simulator.git
cd path-simulator
yarn install
yarn start
```
