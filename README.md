# path-simulator
**path-simulator** is webapp for visualizing the [pathfinding](https://en.wikipedia.org/wiki/Pathfinding) algorithms in realtime. <br>

<br>

## Algorithms

* Dijkstra
* DFS
* 0-1 BFS

<br>

## Build Guide
```bash
git clone https://github.com/AmitHalpert/path-simulator.git
cd path-simulator
yarn install
yarn start
```
